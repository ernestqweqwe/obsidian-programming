### **Псевдоклассы**

Псевдоклассы используются для выбора элементов на основе их **состояния** или **положения** в документе. Они начинаются с одного двоеточия (:).
#### Примеры псевдоклассов:

1. **Состояния пользователя**:
    - :hover — применяется, когда курсор наведён на элемент.
    - :active — когда элемент активирован (например, при клике).
    - :focus — когда элемент находится в фокусе (например, при клике на поле ввода).
    - :visited — для посещённых ссылок.
2. **Положение в структуре**:
    - :first-child — первый дочерний элемент в родителе.
    - :last-child — последний дочерний элемент.
    - :nth-child(n) — выбирает элемент по его порядковому номеру (например, :nth-child(2) — второй элемент).
    - :first-of-type, :last-of-type, :nth-of-type(n) — аналогично, но для элементов определённого типа.
3. **Логические и функциональные**:
    - :not(selector) — исключает элементы, соответствующие селектору.
    - :where(selector) — группирует условия для стилизации.
    - :is(selector) — упрощает выбор нескольких селекторов.
    - :has(selector) — выбирает элементы, содержащие указанный селектор (например, родитель с определённым дочерним элементом).
4. **Другие**:
    - :checked — для отмеченных чекбоксов или радиокнопок.
    - :disabled — для отключённых элементов формы.
    - :empty — для элементов без содержимого.
    - :root — корневой элемент документа (обычно <html>).
